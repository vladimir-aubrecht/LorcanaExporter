# LorcanaExporter
Simple tool for exporting Lorcana cards from official Lorcana TCG aplication.

This webpage is running fully in your browser, no need for any server apart of downloading the backup itself.
Tool will guide you through the process of obtaining backup for official Lorcana TCG application and provides functionality for export.
At this moment only available export options are Lorcana.gg and DreambornInk, but tool is prepared for extending for any other platform if needed.

Feel free to create issue if your platform is missing - I don't promise I will implement it, but I might or somebody from community might do it.
